# Proyecto: Diseño e Implementación de Unidades Organizativas

## Descripción
Este proyecto consiste en el diseño e implementación de un esquema jerárquico de Unidades Organizativas (OU) en Active Directory.

## Entorno de laboratorio
- VirtualBox
- Windows Server
- Active Directory Domain Services

## Dominio creado
corp.local

## Estructura implementada
- Quito
- Guayaquil
- Cuenca

Cada sede contiene usuarios y un grupo de seguridad global.

## Objetivo
Organizar la gestión de identidades por ubicación geográfica y facilitar la administración mediante grupos.
